使用说明

ProxyCLI 支持三种 OAuth Provider，点击上方卡片开始授权流程：

  • Gemini Google Cloud Code Assist - 需要 GCP 项目启用 Cloud AI Companion API
  • Codex OpenAI Codex - 通过 OAuth PKCE 授权
  • iFlow iFlow Chat - 通过 OAuth 获取 API Key

远程访问模式

检测到你正在通过远程地址访问管理面板。点击 Provider 卡片后将弹出引导窗口：
1. 点击链接在新标签页完成 OAuth 登录
2. 登录后浏览器跳转到 localhost（页面无法加载是正常的）
3. 复制地址栏完整 URL 粘贴回引导窗口即可完成认证
